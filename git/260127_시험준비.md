# git and cli

### Cli

- touch 파일생성
- mkdir 새 디렉토리
- ls 현재 디렉토리 내부 파일, 폴더 목록 출력
- cd 디렉토리 변경
- start 파일, 폴더 열기
- rm 파일 삭제
- pwd 현재 작업중인 디렉토리 절대 경로 출력

 ---

### Git

- git init: 로컬 저장소 설정
- git add:  staging area에 추가
- git commit: 해당 시점의 버전을 생성
- git config: --global user.email(name): commit 작성자 정보 설정(생략할 경우 현재 git 저장소에만 적용)
- git status: 현재 로컬 저장소의 파일 상태 보기
- git log: commit 히스토리 보기
- git log --oneline: commit 목록 한줄로 보기
- git config --global -l: git global 설정 정보 보기

- git revert <commit id>: commit id에 해당하는 commit을 없었던 일로 되돌리기
 - git log --oneline 으로 목록을 확인한 후 revert 하면 됨
 - 공백을 사용해 여러 커밋 취소 가능.

- git reset [옵션] <commit id>
 - 특정 커밋으로 되돌아가는 행위
 - 옵션 --soft, --mixed(기본), --hard
 - --soft: 삭제된 commit들의 기록을 staging area에 남김
 - --mixed: 삭제된 commit들의 기록을 working directory에 남김 (기본값)
 - --hard: 삭제된 commit들의 기록을 남기지 않음

 - git restore: Modified 상태의 파일 되돌리기
 - git restore --staged: staging Area에서 Working Directory로 되돌리기
